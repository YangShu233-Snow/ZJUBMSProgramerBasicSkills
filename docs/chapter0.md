# 开始之开始：我们到底要学什么

> 工欲善其事，必先利其器

当你在打开这份文档的时候，是否抱着这样的疑惑：

- 我不知道要从这份文档中学到什么？/ 看完这份文档后我就能搞懂一切了吗？
- 我有点无从下手，因为我不知道能否胜任这份文档所说的东西。
- 看到这里有点困了……我是不是不适合学这个？

哦我的朋友，我是说，伟大的基医er们，不要悲伤，不要害怕，这份文档自被创作之初就怀揣这样的愿望：<b>这是一份轻松、有趣且干货十足的入门文档</b>。它会以轻松诙谐的语调（或者你们已经感受到了，不是吗？）为你讲解一些被前辈们当做常识，但是需要你自己学习了解的入门知识，为你呈上一份详细的，跨越《Python程序设计》至《人工智能基础A》的过渡指南。

接下来，这份文档，或者说“我”，会向你进一步解释一些问题与误区，并向你简单介绍文档的基本特点~

## 这里不教 Python 与 AI

作为入门文档，我并不会在正文中长篇大论 AI 与 Python 相关的知识，前者是你下学期的主要内容，后者是你上学期的学习内容，作为其中桥梁承接的部分，我最多只会提到小部分有关于 Python 项目管理的知识，剩下的内容请参照学过这门课的学长学姐的课程反馈吧~
